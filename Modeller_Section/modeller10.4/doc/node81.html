AllHModel() — prepare to build all-hydrogen models

AllHModel(env, alnfile, knowns, sequence, deviation=None, library_schedule=None, csrfile=None, inifile=None, assess_methods=None)
This creates a new object for building all-hydrogen models. All of the arguments are the same as those for AutoModel().

See section 2.2.5 for an example.
